# NeoBank 🏦

A modern banking dashboard prototype built with Flutter and Supabase. This app allows users to view their total balance, manage credit cards, and track recent transactions in real-time.

## ✨ Features
* **Real-time Database:** Connected directly to Supabase for live updates.
* **Dashboard Overview:** Displays user greetings, total calculated balance, and recent activity.
* **Card Management:** Horizontal scrolling list of user cards with dynamic backgrounds and masked numbers.
* **Transaction History:** Clean, color-coded transaction list showing organization, timestamp, and amount.
* **Add Money / Transfer (UI):** Connected screens for adding funds and viewing card details.

## 🛠 Tech Stack
* **Frontend:** [Flutter](https://flutter.dev/) & Dart
* **Backend / Database:** [Supabase](https://supabase.com/) (Backend-as-a-Service)
